# ⚡ Ясно — AI-планер дня

**Скажи, що в голові — стане ясно, що робити.**

Вивантажуєш хаос думок голосом або текстом → AI розбирає їх на задачі з пріоритетом,
оцінкою часу та дедлайном → сам збирає реалістичний план на сьогодні → режим «Фокус»
веде тебе по плану, блок за блоком, до конфеті 🎉

🔗 **Живий застосунок:** https://day-planner-henna-three.vercel.app/

## Проблема, яку вирішуємо

У голові — каша: десятки думок, справ і дедлайнів. Класичні тудушки вимагають самому
все структурувати (пріоритет, час, порядок) — і тому ними не користуються. **«Ясно»
знімає цю роботу: ти просто вивалюєш думки, а структуру й план робить AI.**

## Наскрізний сценарій (ядро продукту)

1. **Думки** — brain dump: скажи вголос (розпізнавання мовлення uk-UA) або напиши все підряд
2. **AI-розбір** — Claude аналізує текст: окремі задачі, пріоритет (важливо/середнє/не терміново),
   оцінка часу, дедлайн і **час доби** з природної мови («до п'ятниці», «завтра», «о 15:00»)
3. **План на сьогодні** — AI сам відбирає 3–7 справ, які реально варто зробити сьогодні,
   і показує брифінг: скільки задач, скільки часу, з якої почати
4. **Фокус** — веде по плану пачками за пріоритетом: важливе → середнє → дрібниці.
   Прогрес-бар, мікро-свята між блоками, конфеті у фіналі

## Продуктові рішення (чому саме це)

Ми свідомо не «наклали фіч», а вбудували **перевірені техніки продуктивності** так,
щоб кожна підсилювала головний сценарій:

- **AI — ядро, не декорація:** розбір, пріоритезація, план дня, брифінг — усе робить AI
- **Фокус замість списку:** довгий список паралізує — показуємо лише поточний блок справ
- **Реалістичний план:** AI свідомо обмежує день (3–7 задач), решта — у «Вхідних»
- **🐸 Жаба дня** (Eat the Frog): AI підсвічує найважчу задачу — «зроби першою»
- **📋 Розклад дня** (Time Blocking): задачі з часом — як стрічка дня з маркером «зараз»
- **🌅 Розумний ранок:** пропонує додати в план те, у чого дедлайн сьогодні
- **⚖️ Підказка 1-3-5:** м'яко попереджає про перевантаження дня (AI радить — вирішуєш ти)
- **Швидкі жести:** свайп вправо — виконати, вліво — видалити; голосовий ввід
- **Акаунт і синхронізація:** Google-вхід (Supabase Auth), задачі в хмарі (RLS), усі пристрої
- **Mobile-first:** великі тач-зони, темна тема, PWA — можна додати на екран «Додому»

## Стек

- **Next.js 16** (App Router) + React 19 + Tailwind CSS v4 + TypeScript
- **Supabase** — Google OAuth + Postgres із Row Level Security
- **Claude Haiku 4.5** (через OpenRouter) — серверний розбір задач (`app/api/parse`),
  ключ живе тільки на сервері
- **Web Speech API** — голосовий ввід українською, без зовнішніх сервісів
- **Vercel** — деплой, авто-оновлення з кожним push

## Запуск локально

```bash
npm install
# .env.local: NEXT_PUBLIC_SUPABASE_URL, NEXT_PUBLIC_SUPABASE_ANON_KEY, OPENROUTER_API_KEY
npm run dev
```
